About Our Features

About the Desktop Software:
PyBrowse™ is a free private web browser developed by python3.x_dev. PyBrowse™ was created in the year of 2020, using python 3.8.2 and the gtk, webkit, libraries to render web pages, which implements current and anticipated web standards. PyBrowse™ supports diffrent OS's, for more information go HERE. About the Android App:
The mobile app is made with Java and Kotlin in Android Studio v193.66, and is still in development. When it will be released to google play, the websites content will be updated with a link to the playstore app. Thank you for choosing PyBrowse™.
